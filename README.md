## About The Project

![gpp_frontend](readme_media/gpp_frontend.jpeg)

견주들을 위한 굿푸피 앱을 개발합니다.

중요했던 고려 사항들은 다음과 같습니다.

- 사용자 경험적으로 친숙한 색 테마 
- 사용자 시나리오 중심의 기능 및 화면 설계
- 단순한 제어 앱이 아닌 브랜딩을 통한 차별화
